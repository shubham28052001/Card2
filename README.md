## Overview  
This is a clean, responsive card-grid application built with HTML, CSS and JavaScript. It enables users to dynamically filter displayed items by category using a set of interactive buttons. One project variation of card scripting and layout logic.

## Live Demo  
[View it live → Card2](https://shubham28052001.github.io/Card2/)

## Features  
- Filter buttons that show/hide cards by category.  
- Responsive grid layout — adapts from mobile through large desktop screens.  
- Each card showcases an image, title, category, price/description.  
- Simple, lightweight, dependency-free implementation.

## Technologies  
- HTML5  
- CSS3 (Grid, Flexbox, Media Queries)  
- Vanilla JavaScript (ES6)  
- Hosted via GitHub Pages
